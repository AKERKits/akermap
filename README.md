# The AKER map

## Quickstart

### Windows
1. `npm install`
2. `postinstall.bat`
3. `dev.bat`
4. open [http://localhost:8080/webpack-dev-server/](http://localhost:8080/webpack-dev-server/).

### *nix
1. `npm install`
2. `./postinstall`
3. `./dev`
4. open [http://localhost:8080/webpack-dev-server/](http://localhost:8080/webpack-dev-server/).
